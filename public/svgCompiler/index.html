Hilfswerkzeug zum Generieren von animierter Schrift in Form einer SVG-Datei

Diese Seite ist lediglich zur Entwicklung gedacht und sollte nicht auf den Server kopiert werden.

Die Arbeitsschritte sind folgende:

  1. Zeichnen aller Vektoren für alle Buchstaben/Zeichen in Inkscape
  2. Man speichert die Inkscape-Datei als paths_export.svg im gleichen Verzeichnis wie diese index.html.
  3. Man generiert ein JSON aus den Vektoren in Schritt 1
  4. Man kopiert das JSON in die /src/svgCompiler/vectorizer.js und passt die lineHeight an.
  5. Mit hilfe der vectorizer.js-Datei wird der Text, den man im Funktionsaufruf angibt in eine SVG-Datei übersetzt.

Spezifikation der Inkscape-Datei (siehe /src/svg/paths24.svg als Beispiel):

  • Jedes Zeichen, zB der Buchstabe "A", muss als Pfad und nicht als Text in doppelter Ausführung in einer eigen Layer gespeichert werden. Die Layer muss den Namen des Buchstaben tragen, also zB "A". Jeder Buchstabe muss doppelt sein, da dadurch die Breite des Buchstaben ermittelt wird.
  • Der oder die Vektoren für einen Buchstaben müssen in einer eigenen Layer mit dem Namen [Buchstabe] path gespeichert werden, also zB A path.
  • Die so erstellte Inkscape-Datei darf nicht als komprimierte oder optimierte Datei gespeichert werden, sondern als native Inkscape-Datei, da sonst Layernamen entfernt werden.
